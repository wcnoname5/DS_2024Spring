
## 1. `Hilbert.py`

To execute the file, Run `python Hilbert.py --order *int*` for desired curve, use `python Hilbert.py --exec` for generating Hilbert curve from order 1 to 4. `.eps` files would be generated but add `--convert` can convert these files to `.png`. An extra Screen recording video `Hilbert_record.mp4` is provided in the directory in case the convertion is not success.  

## 2. `Hilbert_art.py`

Type `python Hilbert_art.py -file rk.png` for the desired `.eps` file from the `rk.png`. Similarly, add `--convert` to would convert it to `.png`.
